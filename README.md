# 🎓 EduMate
### *Empowering Students through AI-Driven Learning*

**EduMate** is a full-stack, serverless educational platform developed to help students bridge learning gaps. By leveraging **Google Gemini AI** and **Firebase**, it transforms static study materials into interactive courses, adaptive quizzes, and a real-time AI tutoring experience.

---

## 🌟 Key Features

* **🔐 Secure Google Authentication:** Managed via Firebase Auth (OAuth 2.0).
* **🤖 AI Course Architect:** Generates structured learning paths automatically using the **Gemini 1.5 Flash API**.
* **📊 Progress Tracking:** Real-time synchronization of course progress and metadata using Cloud Firestore.
* **🧠 AI-Assisted Simplification:** Breaks down complex academic concepts into digestible, easy-to-understand lessons.
* **🌍 Multilingual Support:** Ability to generate and learn content in over 30 languages.

---

## 🛠 Tech Stack

### **Frontend & Core**
* **Framework:** React.js
* **Language:** **TypeScript** (Strongly typed for better code quality)
* **Styling:** Tailwind CSS & Lucide-React Icons

### **Backend (BaaS - Backend as a Service)**
* **Database:** **Google Cloud Firestore** (NoSQL)
* **Authentication:** Firebase Auth
* **AI Engine:** Google Gemini AI SDK
* **Storage/Logic:** Asynchronous Cloud Operations

---

## 🏗 Data Architecture
The project utilizes a structured **NoSQL Schema** to manage student data. Courses are dynamically generated by AI and stored as sub-collections:
`users / {userId} / courses / {courseId}`

## ⚙️ Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Serji0998/WEB-Project-semester-6.git](https://github.com/Serji0998/WEB-Project-semester-6.git)
    cd WEB-Project-semester-6
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Firebase & Gemini Configuration:**
    Create a `.env` file in the root directory and add your keys:
    ```env
    VITE_FIREBASE_API_KEY=your_key
    VITE_FIREBASE_AUTH_DOMAIN=your_domain
    VITE_GEMINI_API_KEY=your_gemini_key
    ```

4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
